# Network Health Monitoring Dashboard (Minimal)

A lightweight network health monitoring project built using Node.js.

## Features
- Periodically checks endpoint availability
- Measures latency (response time)
- Exposes metrics via REST API
- Visualizes latency using Chart.js
- Basic alert logging for downtime

## Tech Stack
- Node.js
- Express
- Axios
- node-cron
- Chart.js

## How to Run
```bash
npm install
npm start
